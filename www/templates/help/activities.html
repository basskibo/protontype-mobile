Creating activities

Entry of activities i.e. enabled from two different windows in the application, from the start screen, after logging into the application or by clicking the plus icon from the window List of activities.


Entry of working operation

s First it is necessary to select a group of working operations (e.g. Basic processing). After that, it is necessary to select a working operation (e.g. ploughing up to 20 cm depth). Letter search is enabled for easier selection of the working operations by entering text in the search field.


Entry of field

The next step is entry of the field on which the previously selected working operation is done. Upon touching the Add field, fields are displayed where the desired one is selected by touching. For easier selection of the field, it is possible to conduct a search by means of the Magnifying glass icon which is located in the upper right corner. When this icon is clicked, a field for entry of parameters appears based on which the application displays the search results.


The selected field is displayed in the window. Confirmation of selection must be done by touching the icon in the upper right corner.


After the desired field has been selected, the user has an option to select more fields, in unlimited number. Adding each new field is done by touching the Add field option, in an identical procedure as adding of the first field.


In case of a wrong selection, by touching the icon X the user deletes the error i.e. cancels the wrong selection.


After the user selects the required field/s, its/their selection is confirmed by touching the icon in the upper right corner.


Selection of date

It is implied that todayâ€™s date is displayed and if the working operation has been conducted on that day, there is no need to select the start date.

Selection of date is done by touching the date entry option. After that a window is displayed where first the start date is selected by clicking the icon for date selection from the displayed calendar.


Finish data is implied to be the same as the start date and if the activities have been completed on the same date as the start date, there is no need to select the finish date.

If the finish date is different than the start date, it is selected by clicking the field Click here to select the activities finish date. After that the entry field is opened and the date is entered in the same way as the start date.

After the desired start and finish dates have been selected, the selection must be confirmed by touching the icon in the right upper corner.

Selection of materials

Selection of materials starts by touching the selection of materials option.

Then, in the window which appears, it is necessary to touch Add material. After that group of materials must be selected, then sub-groups and the required preparation.

For easier selection of materials, it is possible to conduct a search by means of the Magnifying glass icon which is located in the upper right corner. When this icon is clicked, a field for entry of parameters appears based on which the application displays the search results.

After the material has been selected, it is necessary to enter consumed quantity of the same by touching the field Enter consumed quantity, and confirm the data by touching the icon in the upper right corner.

After the desired material has been selected, the user has an option to select more materials, in unlimited number. Adding each new material is done by touching the Add materials option, in an identical procedure as adding of the first material.

In case of a wrong selection, by touching the icon X the user deletes the error i.e. cancels the wrong selection.

After the user selects the desired material, the selection is confirmed by touching the icon in the upper right corner.

Selection of machine

Selection of machine starts by touching the selection of machine option.

Then, in the window which appears, it is necessary to touch Add machine. After that machine group must be selected, then sub-groups (e.g. Claas Tucano) and the required machine.

For easier selection of machine, it is possible to conduct a search by means of the Magnifying glass icon which is located in the upper right corner. When this icon is clicked, a field for entry of parameters appears based on which the application displays the search results.

After the desired machine has been selected, the user has an option to select more machines, in unlimited number. Adding each new machine is done by touching the Add machine option, in an identical procedure as adding of the first machine.

In case of a wrong selection, by touching the icon X the user deletes the error i.e. cancels the wrong selection.

After the user selects the desired machine, the selection is confirmed by touching the icon in the upper right corner.

Then, in the window which appears, it is necessary to touch Add implement. After that implement group must be selected and the required implement.

Entry of description

The user may add a comment, if he wishes to do so, in the field foreseen for entry of descriptions.

After the data have been entered the activities have been filled in and saved by clicking the icon in upper right corner.

List of activities

By clicking the List of activities a list of all entered activities appears.

It is implied that a list of activities is displayed with the basic data about activities (number, working operation and date). In order to see all the data entered in the activities, it must be touched and then it will be displayed with all entered data.

In this window it is possible to search the activities per their numbers and the name of the working operation, in the search field.

List of activities

Change of an activities is done by selecting the activities one wishes to change from the list of activities.

The activities are opened by touching and the user makes the required changes according to the same principle as with adding of new activities.

After the changes have been made, the activities need to be saved by touching the save icon in the upper right corner and by confirming with the Save button in an additional window which appears.

Activities status

The user determines the status of created activities by clicking the green circle which is located in the upper right corner. After that a window is opened and the user will select the desired option from the list of offered options.

By selecting the options Open, Ongoing and Completed activities has a status which enables its changes.

By locking the activities the user confirms that the activities have been completed and after it is locked, it may not be changed anymore.

Locking is done by having the user select the option Locked and confirm locking by clicking YES in the confirm window.

After selecting the Activities status, the selection must be saved by clicking the save icon, which is located in the upper right corner of the mobile application. By clicking Save in the additional window Activities status is saved.

Deleting activities

Deleting activities is done by selecting the activities one wishes to delete. After touching and holding it an additional window appears where the user must confirm deletion of the activities which are then deleted.
